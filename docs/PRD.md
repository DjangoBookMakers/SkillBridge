# 스킬브릿지 (SkillBridge) PRD 문서

## 제품 개요

스킬브릿지는 ICT를 처음 배우는 예비 개발자를 위한 온라인 교육 플랫폼입니다. 이 플랫폼은 나노디그리(과정) 방식의 학습 체계를 제공하여 사용자들이 단계적으로 개발 역량을 키울 수 있도록 합니다.

## 목표

- ICT 분야 입문자들에게 체계적인 학습 환경 제공
- 실무 중심의 미션과 프로젝트를 통한 실질적 역량 개발
- 학습 진행 상황 추적 및 수료증 발급 시스템 구축
- 강사와 학생 간 상호작용 촉진

## 사용자 유형

1. **수강생**: 플랫폼의 주 사용자로, 과정을 구매하고 학습을 진행함
2. **강사**: 과정 컨텐츠를 생성하고 학생들의 프로젝트를 평가함
3. **관리자**: 플랫폼 전반을 관리하고 컨텐츠 제작 및 등록을 담당함

## 주요 기능

### 1. 사용자 관리

#### 회원가입 및 로그인

- 이메일 기반 일반 회원가입 및 로그인
- 사용자 유형(수강생/강사/관리자)에 따른 권한 관리
- 로그인 후 해당 유형에 맞는 화면으로 리다이렉트

#### 프로필 관리

- 기본 정보 관리: 이름, 이메일, 휴대폰 번호, 생년월일, 성별 등
- 강사 프로필: 자기소개, 경력, 자격사항 관리
- 강사 자격 요청 및 승인 프로세스

### 2. 과정(나노디그리) 관리

#### 과정 구조

- 나노디그리(과정) > 과목 > 강의 > 미션의 계층 구조
- 각 과정은 여러 과목으로 구성 (일반 과목, 중간고사, 기말고사)
- 각 과목은 여러 강의(동영상)와 미션(쪽지시험)으로 구성

#### 관리자/강사 기능

- 과정, 과목, 강의 생성/수정/삭제
- 동영상 업로드 및 관리
- 미션 문제 등록 및 관리
- 중간/기말고사 프로젝트 평가

#### 수강생 기능

- 과정 검색 및 조회
- 과정 구매 및 수강
- 과정별 진행 상황 확인
- 수료증 발급 및 다운로드

### 3. 학습 진행 관리

#### 대시보드

- 관리자 대시보드: 플랫폼 전체 현황 모니터링
- 강사 대시보드: 과정별 학습 진행 상황 확인
- 수강생 대시보드: 개인 학습 진행 상황 및 결과 확인

#### 진행 추적

- 강의별 시청 완료 여부 추적
- 미션(쪽지시험) 결과 기록
- 프로젝트 제출 및 평가 상태 관리
- 과정 전체 진행률 계산 및 표시

### 4. 동영상 학습 시스템

#### 비디오 플레이어

- HTML5 비디오 태그를 활용한 동영상 재생
- 강의 시작으로 시청 완료 처리
- AWS S3 연동을 통한 동영상 저장 및 스트리밍

### 5. 미션 및 평가 시스템

#### 미션(쪽지시험)

- 5지선다 객관식 문제 형식
- 자동 채점 시스템 (80% 이상 정답 시 통과)
- 결과 즉시 확인 가능

#### 프로젝트 평가

- 중간/기말고사 프로젝트 파일 업로드
- 강사의 프로젝트 검토 및 피드백
- 통과 여부 평가

### 6. 수료 인증 시스템

#### 수료 기준

- 모든 강의 시청 완료
- 모든 미션 80% 이상 정답률
- 중간/기말고사 프로젝트 통과

#### 수료증 발급

- 과정 별 수료증 자동 생성
- 사용자 이름 및 수료 날짜 자동 입력
- PDF 형식 다운로드 지원

### 7. 결제 시스템

#### 과정 구매

- 포트원(구 아임포트) API 연동
- 단일 과정 및 여러 과정 동시 결제 지원
- 구매 후 평생 이용 가능

#### 장바구니

- 과정 장바구니 담기/삭제
- 장바구니 내 과정 목록 확인
- 장바구니에서 일괄 결제 기능

### 8. 커뮤니케이션

#### 질의응답

- 강의별 Q&A 기능
- 강사 및 다른 수강생의 답변 기능
- 질문 목록 조회 및 검색

#### 리뷰

- 과정별 평점(1~5점) 및 리뷰 작성
- 리뷰 수정 및 삭제 기능
- 과정 상세 페이지에서 리뷰 목록 확인

## 기술 스택

### 백엔드

- Django
- Django REST Framework
- PostgreSQL (배포) / SQLite (개발)

### 프론트엔드

- Django 템플릿
- Tailwind CSS
- JavaScript

### 인프라

- AWS EC2 (서버 호스팅)
- AWS S3 (동영상 및 파일 저장)
- AWS RDS (데이터베이스)

## 데이터 모델

### 주요 엔티티

1. **User**: 사용자 정보 (수강생, 강사, 관리자)
2. **InstructorProfile**: 강사 프로필 정보
3. **Course**: 나노디그리(과정) 정보
4. **Subject**: 과목 정보 (일반, 중간고사, 기말고사)
5. **Lecture**: 강의 정보 (동영상, 미션)
6. **MissionQuestion**: 미션 문제 정보
7. **LectureProgress**: 강의 진행 상황
8. **MissionAttempt**: 미션 시도 기록
9. **ProjectSubmission**: 프로젝트 제출 정보
10. **Enrollment**: 과정 등록 및 진행 상태
11. **Payment**: 결제 정보
12. **Cart/CartItem**: 장바구니 및 항목
13. **QnAQuestion/QnAAnswer**: 질의응답
14. **CourseReview**: 과정 리뷰

## 사용자 인터페이스

### 주요 화면

1. **메인 화면**: 주요 과정 홍보, 과정 카테고리 필터, 개인화된 추천
2. **과정 상세 화면**: 과정 정보, 강사 정보, 커리큘럼, 리뷰
3. **강의 시청 화면**: 동영상 플레이어, 질의응답, 강의 목차
4. **미션 화면**: 문제 제시, 선택지, 제출 버튼, 진행 상태
5. **대시보드**: 사용자 유형별 맞춤 대시보드
6. **관리자/강사 페이지**: 과정/과목/강의 관리 화면
7. **장바구니/결제 화면**: 장바구니 목록, 결제 정보 입력
8. **프로필 화면**: 개인 정보, 학습 이력, 수료증 목록

## 오류 처리 및 예외 상황

1. **결제 실패**: 오류 메시지 및 재시도 옵션 제공
2. **동영상 로딩 실패**: 대체 화면 및 문제 해결 가이드 제공
3. **네트워크 연결 끊김**: 알림 및 자동 재연결 시도
4. **세션 만료**: 알림 및 재로그인 유도

## 성능 및 보안

- 동시 접속자 100명 이상 지원
- 페이지 로딩 시간 3초 이내
- 동영상 원본 화질 재생 가능

### 보안 요구사항

- 사용자 개인 정보 암호화 저장
- 결제 정보 보안 처리
- CSRF 및 XSS 공격 방지

## 확장성 및 유지보수

### 확장 계획

- 소셜 로그인 통합
- 모바일 앱 개발
- 실시간 화상 강의 기능 추가
- 커뮤니티 기능 강화

### 유지보수 전략

- 모듈화된 코드 구조로 유지보수 용이성 확보
- 충분한 테스트 코드 작성
- 문서화를 통한 개발 과정 기록

## 배포 계획

1. **개발 환경**: 로컬 개발 및 테스트
2. **스테이징 환경**: AWS EC2 테스트 서버 배포
3. **프로덕션 환경**: AWS EC2 프로덕션 서버 배포
4. **CI/CD**: GitHub Actions를 통한 자동화된 배포

## 타임라인

1. **1단계** (2개월): 기본 인프라 및 사용자 관리 시스템 구축
2. **2단계** (3개월): 과정 관리 및 학습 시스템 개발
3. **3단계** (2개월): 결제 시스템 및 수료증 발급 기능 개발
4. **4단계** (1개월): 테스트 및 버그 수정
5. **5단계** (지속): 기능 개선 및 신규 기능 개발

- 사용자 등록 및 유지율
- 과정 완료율

2. **사용자 이탈**: 사용자 참여 유도 기능 및 알림 시스템 구현
3. **결제 시스템 장애**: 백업 결제 시스템 및 수동 처리 방안 마련
4. **데이터 손실**: 정기적인 백업 및 복구 프로세스 수립

## 결론

스킬브릿지는 ICT 입문자들에게 체계적인 학습 환경을 제공하는 온라인 교육 플랫폼으로, 나노디그리 방식의 커리큘럼과 실무 중심 미션을 통해 실질적인 개발 역량을 키울 수 있도록 설계되었습니다. 본 PRD를 기반으로 개발을 진행하며, 사용자 피드백을 반영하여 지속적으로 개선해 나갈 예정입니다.
